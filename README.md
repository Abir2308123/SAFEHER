# SafeHer — Real-Time Safety Monitoring Dashboard

SafeHer is an interactive safety monitoring platform that aggregates live user sentiment 
into dynamic heatmaps and analytics, enabling proactive safety monitoring for campus and 
urban environments.

## Features

- 🗺️ **Live Heatmap** — Aggregates anonymous user reports into a real-time geographic heatmap
- 📊 **Sentiment Analytics** — Tracks safety sentiment trends over time with visual dashboards
- 📍 **GeoJSON Integration** — Custom campus map overlays (SRMIST) using real boundary data
- 🔒 **Anonymous Reporting** — Privacy-first incident reporting with no login required
- ⚡ **Real-Time Updates** — Dashboard refreshes dynamically as new reports come in

## Tech Stack

- **Frontend:** Next.js 14, TypeScript, Tailwind CSS
- **Maps & Geo:** GeoJSON, Custom heatmap rendering
- **Tooling:** ESLint, PostCSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.
